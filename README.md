# Portfolio Website In ReactJS

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/NirajSrivastava18/portfolio
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm start
   ```

4. Open the application in your browser at:
   ```
   http://localhost:3000
   ```
